Jak se připojit

Připojit se může kdokoli – stačí mít Minecraft Java Edition (nebo Bedrock) a postupovat podle kroků níže.

01

Spusť Minecraft Java Edition

Ujisti se, že máš nainstalovaný Minecraft Java Edition (nebo Bedrock - instrukce níže). Můžeš se připojit přes Minecraft verze 1.19 až 1.21.11, ale doporučujeme 1.21.1 - 1.21.11.

-

Jak se připojit přes Bedrock Edition: Postup je dále popsaný, bedrock je nejlepší spustit přes nejnovější verzi.

02

Klikni na „Multiplayer"

V hlavním menu Minecraftu klikni na tlačítko Multiplayer.

-

U bedrocku klikni v hlavním menu na Play, a následně v horní liště vyber záložku servers.

03

Přidej server

Klikni na Add Server (nebo „Přidat server"). Do pole Server Address zadej přesně tuto adresu:

gymjimc.org

Na bedrocku klikni na + Add Server (nebo „Přidat server"). Do pole Server Address zadej opět gymjimc.org. V kolonce port musí být 19132.

04

Připoj se a hraj!

Klikni Done (na bedroku na Add server), server se zobrazí v seznamu. Dvojklikem se připoj. Přijdou instrukce přímo ve hře. Užij si hru!

ℹ

Máš problém s připojením? Zkus restartovat Minecraft nebo zkontroluj, zda máš správnou verzi. V případě problémů nám napiš na mail minecraft@jirovcovka.net!

Voicechat

Na serveru běží Simple Voice Chat mod – můžeš mluvit s ostatními hráči přímo ve hře pomocí mikrofonu, prostorově podle polohy.

Aby voicechat fungoval, musíš si nainstalovat mod do svého Minecraftu. Vyber verzi podle svého modloaderu:

Fabric Doporučeno

Pro Fabric modloader (verze 1.21.1). Potřebuješ oba soubory níže:

⬇ Simple Voice Chat (Fabric) ⬇ Fabric API (povinné)
Forge

Pro Forge modloader (verze 1.21.1).

⬇ Simple Voice Chat (Forge)

Jak nainstalovat mod

  1. Stáhni příslušný .jar soubor (nebo oba pro Fabric).
  2. Otevři složku .minecraft/mods – na Windows najdeš přes %appdata%\.minecraft\mods.
  3. Přesuň stažený soubor (nebo soubory) do složky mods.
  4. Spusť Minecraft přes Fabric/Forge launcher a připoj se na server.
  5. Ve hře zmáčkni V pro nastavení voicechatu.
💬

Voicechat je volitelný – bez modu se na server normálně připojíš, jen neuslyšíš ostatní a oni neuslyší tebe.
